This project aims to build a simple file copy tool called mycp using C language and Linux system calls. It helps students understand how files are handled at the system level.

This project involves implementing a simplified version of the cp command in C using system calls like open, read, write, and close. It also includes Bash scripts to test the implementation and analyze system calls and memory management using valgrind.
